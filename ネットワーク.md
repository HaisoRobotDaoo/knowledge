# ネットワーク

## pingの確認

```bash
ping <ip>
```

## ssh接続

```bash
ssh <user_name>@<ip>
```

## トピックHz

次のコマンドで、データが公開される速度を確認することもできます。

```bash
ros2 topic hz <topic_name>
```

このレートは、`ros2 topic hz`コマンドによって作成されたサブスクリプションの受信レートを反映します。これは、プラットフォーム リソースや QoS 構成の影響を受ける可能性があり、パブリッシャー レートと完全に一致しない場合があります。

## トピックbw

次のコマンドで、トピックで使用される帯域幅を表示できます。

```bash
ros2 topic bw <topic_name>
```

帯域幅は、`ros2 topic bw`コマンドによって作成されたサブスクリプションの受信速度を反映します。これは、プラットフォーム リソースや QoS 構成の影響を受ける可能性があり、パブリッシャーの帯域幅と完全に一致しない場合があります。
