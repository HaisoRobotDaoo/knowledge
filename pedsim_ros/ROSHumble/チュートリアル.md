### ワークスペースの反映と環境変数の設定

```bash
cd ~/pedsim_ws/
source /opt/ros/humble/setup.bash
source ~/pedsim_ws/install/setup.bash
export GAZEBO_MODEL_PATH=/home/et/pedsim_ws/install/pedsim_gazebo_plugin/share/pedsim_gazebo_plugin/models:/usr/share/gazebo-11/models:/opt/ros/humble/share/turtlebot3_gazebo/models
export GAZEBO_RESOURCE_PATH=/usr/share/gazebo-11
export GAZEBO_PLUGIN_PATH=/usr/lib/x86_64-linux-gnu/gazebo-11/plugins
```

### Turtlebot3 の使用例

```bash
ros2 launch pedsim_gazebo_plugin gazebo_tb3_house_demo_launch.py
```
