## Terminator
以下は端末で実行してください。
インストール後、Terminatorに切り替えてください。

```bash
sudo apt update
sudo apt install terminator
```

### コマンド解説♪
* `sudo` (super user do) = 管理者（root）権限で実行
* `apt` (advanced package tool) = ソフトの入手先（リポジトリ・倉庫）

`sudo apt update` は管理者権限でaptリポジトリに対して最新版（update）があるか確認しています。
すなわち、PCと最新情報の関連付けともいえます。毎回打ち込む必要はありません。
定期的にupdateをするんだなということを頭に入れておいてください。

## Jetbrains Tool Box
（PyCharmのみ、VScodeがいい人は別途調べて）

公式サイト
* JetBrains Toolbox App：ツールを簡単に管理

```bash
# こんなだるかったっけ？
mkdir -p ~/opt
cd ~/opt
cp ~/ダウンロード/jetbrains-toolbox-*.tar.gz .
tar -xzf jetbrains-toolbox-*.tar.gz
cd jetbrains-toolbox-*/
./bin/jetbrains-toolbox
rm -f ~/ダウンロード/jetbrains-toolbox-*.tar.gz
rm -f ~/opt/jetbrains-toolbox-*.tar.gz
```
