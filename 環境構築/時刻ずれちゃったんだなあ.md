# 時刻ずれちゃったんだなあ

## Linux-Windows間で時刻を同期する
端末で以下のコマンドを実行してください。

```bash
sudo timedatectl set-local-rtc 1
```

---

### チョットカシコク：なぜするの？

LinuxとWindowsをデュアルブートする際に、タイムゾーンがLinuxは**UTC基準**であるのに対し、Windowsは**ローカルタイム基準**であるため、日本時刻（Windows）と比べて**-9時間**のずれが起きてしまいます。

これを防ぐため、時刻設定を行います。
