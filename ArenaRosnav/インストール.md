Arena 4.0ドキュメント
Arena5.0はROS2 Humble未対応、自動インストール未対応であるため、今回は推奨バージョンのArena4.0をインストールします。
[Arena 4.0](https://arena-rosnav.readthedocs.io/en/latest/)
インストールガイド
！！注意事項！！重要！！
Python環境が現在ソース化されていないことを必ず確認してください。ros2 がインストールされている場合は、ソース化されていないことを必ず確認してください。
ネイティブ Ubuntu 22.04 を公式にサポートしています。WSL 上の Ubuntu も限定的なサポート（Gazebo のみ）で動作します。
自動インストールにはcurl（カール）というものを利用します。
```bash
sudo apt install curl
```

自動インストール

```bash
curl https://raw.githubusercontent.com/Arena-Rosnav/arena-rosnav/humble/installers/install.sh > install.sh
bash install.sh
```


インストーラスクリプトはフォールトトレラント（ネットワークの不具合、メモリ不足など）に対応しています。インストールに失敗した場合は、スクリプトを再実行すると、前回の時点から再開されます。
「インストールが完了しました」というメッセージが表示されたら、インストールは成功です。
オプションの依存関係
デフォルトでは、ROS 2 と Arena エコシステムのみがインストールされます。
インストールの最後に、インストーラーはオプションの依存関係（現在はGazebo、Isaac、planners）をインストールするかどうかを尋ねます。これらの依存関係を後でインストールしたい場合は、ワークスペースのルートディレクトリからインプレースインストーラーを実行し
```bash
bash src/arena/arena-rosnav/installers/install.sh
```

、プロンプトが表示されるまでお待ちください。
注意事項
ROS 2はワークスペース内でコンパイルされます。ROS 2とArenaエコシステムは大規模です。少なくとも80GBのストレージ容量が必要です。インストールには1時間以上かかる可能性があります。ArenaはUbuntu 22.04でのみ動作します。
