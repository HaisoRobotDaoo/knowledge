インストールガイド
Jazzyは、Gazebo Classicに代わる新しいGazeboモダンシミュレータを導入しました。そのため、Jazzy以降では、適切なパッケージを導入するためのインストールパッケージと手順が若干異なります。
公式ドキュメントに記載されているように[ROS 2バイナリパッケージ](https://docs.ros.org/en/rolling/Installation/Ubuntu-Install-Debians.html)をインストールします。
オペレーティング システムのパッケージ マネージャーを使用して Nav2 パッケージをインストールします。

```bash
sudo apt install ros-humble-navigation2
sudo apt install ros-humble-nav2-bringup
```


gazebo 用のデモ ロボット (Turtlebot) をインストールします。
Iron 以前では、Gazebo Classic 用の Turtlebot 3 パッケージをインストールします。
```bash
sudo apt install ros-humble-turtlebot3-gazebo
```
